# Bug_Tracking_System
 This project is a simple implementation of a bug tracking software.

Stack
- .NET Core MVC C#
- .NET Core WEB API
- MS SQL Server

Functionalities
- Users can add, edit, and remove tickets.
- Users can add, edit, and remove issues.
